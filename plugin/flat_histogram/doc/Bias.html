Bias

class Bias

Bias for flat histogram Monte Carlo. Assumes a one-dimensional macrostate.

CriteriaWriter outputs the following as a row for each Macrostate:

  • ”ln_prob” the LnProbability.

  • ”delta_ln_prob” the change in LnProbability from the previous Macrostate (e.g., ln_prob(state) - ln_prob(state - 1), so the lowest state is NaN).

Subclassed by feasst::TransitionMatrix, feasst::TransitionMatrixGuess, feasst::WLTM, feasst::WangLandau
